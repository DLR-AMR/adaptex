Parameters
[in]vertexAn array of size at least dim giving the coordinates of the vertex to interpolate
[in]corner_valuesAn array of size 2^dim * 3, giving for each corner (in zorder) of the unit square/cube its function values in 3D space.
[out]evaluated_functionAn array of size 3, on output the function values at vertex are stored here.
