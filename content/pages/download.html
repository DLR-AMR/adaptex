TiGL runs on a Windows, Linux, macOS and Android. We support a range of Linux distributions, including Ubuntu and CentOS. TiGL supports Python 3.5 and above. See more details in the user guide.

Open Source

TiGL is Open Source, published under the Apache 2 License, and uses other Open Source software.

Windows

On Windows, a single zip file is provided to set up client and server installations.

Download

macOS

On macOS, we provide a apple disk image (DMG) file. To install, drag TiGL Viewer into your app folder.

Download

Linux

Linux packages can be installed directly from the open build service. Alternatively, TiGL can be installed locally using a script.

Install Install-script
Source Code

Clone or fork the source code from github to build or modify TiGL by yourself.

Python

TiGL and all it's dependencies can be installed best with the conda package manager as follows:

conda install -c dlr-sc tigl3
